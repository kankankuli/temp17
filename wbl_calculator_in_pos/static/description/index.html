Work Flow of POS Calculator:

Detailed Features and Steps:

  • Odoo doesn't come with a built-in calculator for its Point of Sale system (POS) by default.
  • Consequently, POS users need to use external software or other tools when they want to make calculations while billing customers.
  • Our "ADD POS CALCULATOR" app for Odoo solves this problem by adding a calculator to the POS app, making it effortless for POS employees to perform calculations.
  • This app is compatible with multiple shops, allowing the admin to choose specific shops or sessions where they want the calculator to be active or inactive.
Benefits of POS Calculator:

User-Specific Discount Rules

  • The core functionality of the app allows the admin to create user-wise discount rules for salespersons.
  • These rules enable precise control over the discounting capabilities of each salesperson within the Odoo Sales module.

Bulk Action Discount Configuration

  • Admins can efficiently set discount rules in bulk, saving time and effort when managing multiple salespersons.
  • Bulk configuration ensures consistency and accuracy in discount rule application across the sales team.

Enhanced Sales Control

  • By implementing user-wise discount rules, the app provides administrators with enhanced control over the sales process, preventing excessive or unauthorized discounting.

Real-Time Discount Application

  • Discounts are applied in real-time during the quotation creation process, ensuring that salespersons adhere to the predefined rules instantly.

Intuitive Admin Interface

  • The app features an easy-to-use administrative interface that simplifies the process of configuring and managing user-wise discount rules.

Customized Sales Approach

  • The app allows businesses to implement a customized sales approach by tailoring discount rules according to the capabilities and responsibilities of individual salespersons.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise (On-premise and Odoo.SH) as well as Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
